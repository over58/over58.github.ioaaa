### client

| clientTop    | border-top-width           |
| ------------ | -------------------------- |
| clientLeft   | border-left-width          |
| clientWidth  | 水平方向 content + padding |
| clientHeight | 竖直方向 content + padding |

### offset

| offsetTop    |                                     |
| ------------ | ----------------------------------- |
| offsetLeft   |                                     |
| offsetWidth  | 水平方向 content + padding + border |
| offsetHeight | 竖直方向 content + padding + border |
